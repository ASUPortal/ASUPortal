Как сформировать шаблон печатной формы документа

	описание
шаблон представлен в формате .rtf с элементами специальной разметки для указания места размещения импортируемых в шаблон данных при формировании результирующего документа-печатной формы.
	
	порядок формирования шаблона
1. подготовить образец в формате rtf 
2. расставить специальные символы разметки в файле rtf, например {var_name}
3. связать указанные переменные с php-файлом генерации данных, т.е. сопоставить метку {var_name} некоторому описанию (текстовому, графическому, табличному или др.)	
4. проверить результат генерации


	примечания
вставку меток разметки лучше выполнить из предварительно подготовленного txt-файла, т.к. при наборе меток в файле rtf возможен визуально скрытый импорт спец.символов, которые помешают произвести корректное сопоставление метки и ее описания при формирования результата генерации.
