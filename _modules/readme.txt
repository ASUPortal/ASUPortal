Модули портала, список, специальная версия портала для разработки
--------------------------------------------------------------------------------------------------
Типовой модуль включает:
-файл интегратор (с указанием пути к порталу и необходимым библиотекам)
-персональные модули php (вложенные модули рекомендуется называть .inc)
-sql-код БД (если используется)
-презентацию по возможностям и использованию модуля
-дополнительные js,css, img файлы

При разработке модуля рекомендуется к использованию облегченная версия портала АСУ - Портал - версия разработчика.


--------------------------------------------------------------------------------------------------

Руководство развертывания и использования портала

	О системе
Версия разработчика является усеченной версией образовательного портала и предназначена для упрощения процесса разработки модулей задач портала. С полноценной версией Вы можете ознакомиться на asu.ugatu.ac.ru или asu-ugatu.ueuo.com.
В этом режиме авторизация выключена, доступна тестовая задача дипломные проекты , как пример для реализации с учетом особенностей интеграции 
Дополнительно выключены все службы по распределению доступа к задачам 

	Требования к ПО перед установкой
приложения является веб-ориентированным и требует для своей работы Apache 2+, MySQL 5+

	Установка
Поставка содержит исходные коды портала на языке PHP, sql-код базы данных asu_demo.sql.zip. Скопируйте файл в папку проектов веб-сервера и создайте БД в MySQL на основе sql-кода базы данных. Внесите изменения в файл sql_connect.php
