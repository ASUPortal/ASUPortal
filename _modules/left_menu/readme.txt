	модуль настройки и отражения главного меню


для предварительного знакомства просмотрите файл "пример-отражения.mht" для демонстрации возможностей левого меню

	сведения о файлах
данные меню		menu.xml
настройка параметров	index.php

	переработан компонент sdMenu
-добавлена возможность подстветки ссылки меню по части адреса ( по умолчанию - при точном совпадении)
-возможность создания разделов без дочерних элементов
-интеграция с xml средствами php (не требуется доп.библиотека)

	особенности
для внутр.портала (PHP 5) в файле parse.php расскоментировать строку
$out=iconv("utf-8", "windows-1251", $out);
иначе меню отображено не будет
