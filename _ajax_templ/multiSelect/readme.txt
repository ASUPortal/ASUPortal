	описание
модуль реализован на jQuery с использованием плагина расширения multiSelect
автор интеграции Агапов Р.Н.

	назначение
-возможность группировки записей сразу по нескольким критериям одного списка
-изменение формы отражения списков множественного выбора (Multi-select) для компактного размещения на экране и печати

	использование
-применяется в "Табель сотрудников" (kadri_timetable.php)
-при использовании требуется указать ссылки на:
1. jquery-1.3.2.min.js
2. jquery.inlinemultiselect-1.2.min.js 
3. styles.css
-дополнительно встроить вызов в вызываемом файле
